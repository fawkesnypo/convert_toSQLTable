Olá!

Essa API tem como finalidade converter uma planilha(CSV, XLS, XLSX) em um código SQL.
Ela possui três requisições POST que retornaram um caminho para download de um arquivo .sql!

Routes Parâmetros Função
/CREATE_TABLE name - Nome da tabela
file_dir - Diretório da planilha 		Gerar um arquivo .sql contendo o comando "CREATE TABLE" do SQL de acordo com as colunas presentes na planilha.
/INSERT_TABLE name - Nome da tabela
file_dir - Diretório da planilha 		Gerar um arquivo .sql contendo o comando "INSERT" do SQL de acordo com as inforpresentes na planilha.
/CREATE_AND_POPULATE name - Nome da tabela
file_dir - Diretório da planilha 		Gerar um arquivo .sql contendo o comando "CREATE TABLE" e "INSERT" do SQL de acordo com as informações presentes na planilha.
